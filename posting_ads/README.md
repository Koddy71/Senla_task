Система размещения частных объявлений.

Архитектура проекта
User Service — регистрация, профили, аутентификация, роли
Ad Service — управление объявлениями (CRUD), поиск, фильтрация, история продаж, рейтинг продавцов
Payment Service — обработка платного продвижения объявлений (Имитация платежа)
Messaging Service — личная переписка, комментарии к объявлениям
Rating Service — расчёт рейтинга продавцов (на основе отзывов/успешных сделок).
API Gateway — единая точка входа, маршрутизация, проверка JWT
Service Discovery (Eureka) - регистрация и обнаружение сервисов

Роли и доступ (User – Role - Permission)
Неавторизованный пользователь:
•	просматривать объявления
•	искать и фильтровать объявления
•	открывать карточки объявлений
•	просматривать рейтинг и историю продаж продавцов

Авторизованный пользователь:
•	всё что может неавторизованный пользователь
•	редактировать профиль
•	создавать объявления
•	редактировать и удалять свои объявления
•	оставлять комментарии
•	вести личную переписку
•	оплачивать продвижение объявления

Модератор:
•	всё что может пользователь
•	удалять и изменять объявления и комментарии

Админ:
•	выполнять все операции в системе
•	изменить роли пользователей 
•	удалить/заблокировать пользователя
